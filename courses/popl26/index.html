Principles of Programming Languages

Spring 2026

This is an advanced graduate course on programming languages. The primary goal of this course is to provide students with a more modern and sophisticated perspective on languages and programming practice. This year, this course will focus on languages and programming for heterogeneous systems.

Potential topics include (but not limited to):

  • heterogeneous architectures: GPU, TPU, NPU, among others.
  • Low-level interfaces: CUDA, Hip
  • High-level interfaces: especially Python-based DSLs like Triton, cuTile, Mojo
  • Parallel programming : principle and practice
  • Security: and potential mitigations.

Prerequisite: basic C and Python.
