NexaC: Design and Implementation of the Next Generation GPU Compiler

Overview:

With the exponential growth in technology and data, traditional CPUs have increasingly grappled with limitations in addressing compute-intensive tasks. While originally designed for graphic rendering, Graphics Processing Units (GPUs) possess an innate parallelism architecture, making them highly effective for non-graphics parallel computations. This realization has given rise to the widespread study of General-Purpose Graphics Processing Units (GPGPUs). Over recent years, a multitude of studies and real-world applications have underscored the immense value of GPGPUs in both academic research and the industrial sector. Fields such as machine learning, deep learning, computational fluid dynamics, and bioinformatics, to name a few, have witnessed the transformative potential of GPGPUs. These units have empowered scientists and engineers to tackle problems that were previously deemed intractable.

The DexaC initiative endeavors to address these prominent gaps, placing a distinct emphasis on the enhancement of GPGPU compiler research, encompassing architectural design, algorithmic optimization, and robust security mechanisms. To be specific, this project aims to: :

  • Fuzz Testing for Evaluating Robustness and Security of LLVM-based GPGPU Compilers
  • Post-link Binary Optimization for GPGPUs: Extending BOLT for Enhanced Parallel Computing Performance
