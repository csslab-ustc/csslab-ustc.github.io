昇腾自动驾驶领域算子开发-算子实现示例

作为昇腾自动驾驶领域算子预研工作的一部分，尝试了对几个算子进行实验性的移植 （暂未开展针对昇腾芯片的端到端功能验证及算子级优化等）。

实验性移植的算子如下所示：

1）ball query

源算子链接：ball_query_cuda.cu

移植代码：ball_query.cpp

2）gather points

源算子链接：gather_points_cuda.cu

移植代码：gather_points.cpp

3）ingroup index

源算子链接：ingroup_inds_kernel.cu

移植代码：ingroup_index.cpp
